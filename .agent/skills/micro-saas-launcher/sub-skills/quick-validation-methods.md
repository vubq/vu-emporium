# Quick Validation Methods

1. **Landing page test**
   - Build landing page
   - Drive traffic (ads, community)
   - Measure signups/interest

2. **Pre-sale**
   - Sell before building
   - "Join waitlist for 50% off"
   - If no sales, pivot

3. **Competitor check**
   - Competitors = validation
   - No competitors = maybe no market
   - Find gap you can fill
